# qqmusicapi
VUE QQMUSIC后台程序

## 现有功能

- 榜单api
  - /toplist
  - GET
- 推荐api
  - /recommend
  - GET
- 搜索api
  - /search
  - GET
  - 参数1: name
- 歌曲信息api
  - /musicdetail
  - GET
  - 参数1: id
- 歌词信息api
  - /musiclyric
  - GET
  - 参数1: id
